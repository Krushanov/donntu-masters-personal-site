    1. Разработка защищенного протокола обмена сообщениями между пользователями и клиент-серверного программного обеспечения Crypto Messenger

      Авторы: А.И. Крушанов, А.В. Чернышова

      Описание: В данной работе протокол обмена мгновенными сообщениями на базе протокола XMPP, сформированы требования к клиент-серверному программному продукту Crypto Messenger. Проведено проектирование и реализация программного продукта.

      Источник: Программная инженерия: методы и технологии разработки информационно-вычислительных систем (ПИИВС-2018) / Материалы II Международной научно-практической конференции. – Донецк, ДонНТУ – 2018.

    2. Модели для автоматизированной оптимизации производительности систем управления базами данных

      Авторы: А.А. Блажко, А.Ю. Левченко, А.С. Пригожев

      Описание: В работе рассматриваются методы повышения производительности информационных систем, построенных на базе СУБД. Предлагается использовать метод настройки параметров СУБД для сокращения времени ответа и уменьшения ресурсоемкости операций по настройке СУБД. Рассмотрена архитектура и алгоримт построения операционного профиля БД.

      Источник: А.А. Блажко, А.Ю. Левченко, А.С. Пригожев. Модели для автоматизированной оптимизации производительности систем управления базами данных / Журнал радіоелектронні і комп'ютерні системи № 7, Одеса, 2010, с. 24-29, http://www.irbis-nbuv.gov.ua ...

    3. Алгебраический подход к оптимизации разработки и эксплуатации систем управления базами данных

      Авторы: Е.П. Емельченков, Н.А. Левин, В.И. Мунерман

      Описание: В статье рассматривается автоматизация программированиязадач обработки больших объемов информации в базах данных. Предложен специальный тип объектов — универсальная алгебраическая машина, припомощи которого доказаны изоморфизм моделей и возможностьих параллельной реализации.

      Источник: Е.П. Емельченков, Н.А. Левин, В.И. Мунерман. Алгебраический подход к оптимизации разработки и эксплуатации систем управления базами данных / Системы и средства информ., 2009, дополнительный выпуск, с. 114-137, http://www.mathnet.ru ...

    4. Информационная безопасность распределенных информационных систем

      Авторы: В.Ю. Статьев, В.А. Тиньков

    5. Классификация угроз в системах управления базами данных

      Авторы: Д.Р. Утебов, С.В. Белов

      Описание: В работе приводится классификация угроз в системах управления базами данных. Классификация составлена на основе обобщенной схемы классификации угроз, предложенной В.А. Герасименко. Описаны внешние и внутренние угрозы для баз данных. Угрозы разделены на три уровня: уровень сети, уровень операционной системы, уровень базы данных.

      Источник: Д.Р. Утебов, С.В. Белов. Классификация угроз в системах управления базами данных / Вестник Астраханского государственного технического университета № 1, Астрахань, 2008, с. 87-92, https://cyberleninka.ru ...
